About the Project

This Java-based Spring Boot application implements a minimalistic R-tree index using SQLite. The R-tree is used for efficient spatial sorting by distance to a query point, making it a powerful tool for spatial data queries.

It's designed to be simple yet effective, enabling fast querying of data points based on proximity without requiring a full spatial database.

How It Works

The application leverages an R-tree index, a hierarchical data structure commonly used for spatial data, to partition space and speed up searches. In this project, it's applied for spatial queries, specifically for sorting points by distance.
